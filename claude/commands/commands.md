## 🎯 コマンドの追加方法

新しいコマンドを追加したい場合は、以下のディレクトリにMarkdownファイルを作成してください：

```
/Users/akkin/.claude/commands/your-command-name.md
```

コマンド名がファイル名（拡張子なし）になります。例：
- `projects.md` → `/projects` コマンド
- `commands.md` → `/commands` コマンド

## 💡 便利なコマンドアイデア

今後、以下のようなコマンドを追加すると便利かもしれません：

- `/deploy` - デプロイ手順を表示
- `/setup` - セットアップ手順を表示
- `/db` - データベース操作のガイド
- `/api` - API設定のガイド
- `/help-project` - プロジェクト固有のヘルプ

コマンドを追加したい場合は、「〇〇のコマンドを作って」と依頼してください。
